# Install git
```Windows
scoop bucket add navine-cli-oss https://github.com/NavineDevs/navine-cli-oss
scoop install navine-cli-oss
scoop install jid
navine-cli-oss
```

# Delete
```Windows
scoop uninstall navine-cli-oss
scoop cache rm navine-cli-oss
scoop update
```
# Reinstall
```Windows
scoop uninstall navine-cli-oss
scoop cache rm navine-cli-oss
scoop update
scoop install navine-cli-oss
navine-cli-oss
```
